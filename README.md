Автотесты для учебного сервиса "Яндекс.самокат".

Dictionary
1.home_page_dictionary.py - ожидаемые ответы на главные вопросы
2.main_page_dictionary.py - ссылки для проверки функциональности сервиса
3.order_page_dictionary - заполнение полей для заказов

Locator
1.home_page_locators.py - локаторы для куки, кнопок заказать, вопросов, отвнтов на главной странице учебного сервиса
2.main_page_locators.py - локаторы для длготипов сервиса "Яндекс" и "Самокат"
3.order_page_locators.py - локаторы для полей в формах заказа, локаторы для кнопок далее и заказать

Pages:
dzen_page.py - функция загрузки страницы дзена
home_page.py - функции для принятия куки, сверки вопросов, поиска кнопок и логотипов и клика на них
main_page.py - основные функции для всего сервиса и проекта
order_page.py - функции для перехода в форму заказа заполнения полей и клика на кнопки

Tests:
test_home_page.py - тесты вопросов, кнопок и доготипов главной страницы
test_order_page.py - тесты в формах заказа

allure_results - отчеты о тестировании
